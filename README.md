# 🏈 BURWINKEL 2K: Math Football

**BURWINKEL 2K** is a high-energy, football-themed educational web game designed to help students master quadratic equations and algebraic forms. By answering math questions under a time limit, players advance down the field, score touchdowns, and defend their home turf against the opposing "Algebra" team!

This interactive application is completely self-contained within a single HTML file, making it incredibly easy to host, share, or play offline.

## 📖 Description

In BURWINKEL 2K, students take on the role of the play-caller. The "Wildcats" are on offense, and every correct answer gains yardage or scores points, while incorrect answers or running out of time lead to turnovers. The game drills students on recognizing different forms of quadratic equations (Standard, Factored, Vertex/Other) and matching equivalent expressions.

It features a full arcade-style presentation with custom retro typography, smooth animations, an interactive heads-up display (HUD), and a virtual play clock to keep the pressure on!

## ✨ Features

- **Dynamic HUD & Scoreboard:** Tracks the current score, quarter, down, distance, and play clock—just like a real football game broadcast.
- **Math Mechanics:**
  - **Form Recognition:** Quickly identify if an equation is in Standard form, Factored form, or another form.
  - **Equation Grouping:** Select and group matching or equivalent equations to execute a "play".
- **Arcade Visuals & Animations:** Built-in CSS animations, interactive UI buttons, confetti celebrations, and a dynamic football field background.
- **Audio & Sound Mixer:** Features an interactive sound settings panel to toggle background music and sound effects (if linked).
- **Keyboard & Mouse Support:** Fast-paced play with fully mapped keyboard hotkeys (1, 2, 3 or A, B, C) alongside standard point-and-click functionality.
- **Commentary Ticker:** A bottom-screen scrolling ticker provides playful feedback and game status updates.

## 🎮 How to Play

1. **Start the Game:** Click anywhere on the title screen to kick off.
2. **Beat the Play Clock:** You have a limited amount of time to make a play before getting a "Delay of Game" penalty.
3. **Call the Play:**
   - For classification questions, look at the equation on the screen and choose its correct mathematical form.
   - For grouping questions, select the equations that match or share the requested properties and confirm your play.
4. **Drive down the Field:** Correct answers earn First Downs and Touchdowns. Incorrect answers result in Interceptions or Turnovers on Downs!
5. **Win the Game:** Score more points than the Algebra team before the game ends!

## 🛠️ Technologies Used

- **HTML5:** Semantic structure for the UI and Heads-Up Display.
- **CSS3:** Advanced animations, flexbox/grid layouts, gradients, and custom retro fonts via Google Fonts (`Bebas Neue`, `Oswald`, `Rajdhani`).
- **Vanilla JavaScript:** Game loop logic, math problem generation/checking, timer functionality, and DOM manipulation. No external libraries or frameworks are required.

## 🚀 Setup & Installation

Because the entire game is contained in one file, no build tools or servers are required.

1. **Download** the `index.html` file.
2. **Open** the file directly in any modern web browser (Chrome, Safari, Firefox, Edge).
3. *Note: An active internet connection is recommended the first time you play to ensure the custom Google Fonts load correctly.*

## ✏️ Customization (For Teachers/Developers)

If you'd like to adapt the game for your own classroom:
1. Open the `.html` file in a text editor.
2. Locate the question bank arrays in the JavaScript block.
3. Add your own equations, formulas, or vocabulary words. You can easily adapt the text to cover different math topics, history facts, or language translations while keeping the football game logic intact!
